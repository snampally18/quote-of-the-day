# Quote of the Day

A simple website that gives you a random motivational quote and a "focus" to carry with you today.

**Live site:** https://todays-quote-focus.netlify.app

## How it works

- Click the button to reveal your quote, its author, and a focus word for the day
- Each visitor gets their own random quote from a bank of 90 curated quotes
- That quote stays the same if you come back later the same day (saved in your browser), and changes the next day

## Built with

Plain HTML, CSS, and JavaScript — no frameworks, no backend, no build step. Just three files:

- `index.html` — page structure
- `style.css` — styling
- `script.js` + `quotes.js` — quote logic and the quote bank

Hosted for free on [Netlify](https://netlify.com), deployed automatically from this repo. Visitor analytics via [Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/).
